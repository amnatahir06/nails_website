# 💅 Nails Website

A modern and responsive website designed for a nail salon.  
This project presents a clean and visually appealing interface where users can explore nail services, view pricing, and experience the salon's offerings online.

The goal of this project is to demonstrate front-end web development skills through a stylish and user-friendly design suitable for beauty and salon businesses.

- Fully responsive layout
- Service showcase section
- Pricing display
- Smooth navigation
- Clean and organized design
- Suitable for beauty and salon businesses

This project was built using:

- **CSS3** – for styling and layout
